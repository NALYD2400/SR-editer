Démarrer

  1. Télécharge l’installateur Windows depuis la page d’accueil.
  2. Crée un compte (e-mail ou Discord) sur login.html.
  3. Ouvre SR Editer et connecte-toi avec le même compte.
  4. Ouvre un dossier de mods extrait (fichiers YDR + YTD, ou OpenFormats ODR).
Prérequis : Windows 10/11 x64, ~8 Go RAM, GTA V Legacy et/ou FiveM. Le rendu et le décodage restent en local.

Interface

L’app est organisée en zones :

  • Explorateur — fichiers, RPF, dossiers de mods.
  • Texture Studio — édition 2D (calques, masques, pinceaux).
  • Viewport 3D — prévisualisation live du mesh.
  • Bibliothèque — textures cloud prêtes à appliquer.
  • Réglages / Compte — thème, abonnement, administration.

Texture Studio

Édite une texture embarquée ou externe :

  • Ouvre un drawable / fragment et sélectionne une texture.
  • Travaille en calques (peint, gomme, opacité, masque alpha).
  • Exporte en PNG / DDS selon ton offre.
  • L’aperçu 3D se met à jour pendant l’édition.

Bonnes pratiques

  • Garde toujours un backup : l’app crée une sauvegarde avant écriture directe.
  • Travaille d’abord sur une copie du mod FiveM.
  • Vérifie la résolution (HD / 4K selon ton plan).

Explorateur RPF

Parcours :

  • l’installation GTA V,
  • les ressources FiveM,
  • des archives / dossiers locaux.

Tu peux ouvrir des assets RAGE (YTD, YDR, YFT, YDD) et des formats OpenFormats. Le moteur RSC7 décode nativement — pas de faux aperçu.

Viewport 3D

Le viewport WebGL affiche le modèle et synchronise les textures éditées. Utilise-le pour valider les UV, les seams et le rendu avant d’exporter / d’installer le mod.

Bibliothèque de textures

La bibliothèque cloud contient des textures classées (métal, bois, tissu, néon…).

  • Dans l’app : section Bibliothèque → appliquer sur le projet.
  • Sur le site : page Bibliothèque pour parcourir et télécharger.
Accès bibliothèque (site) : connexion obligatoire. Sans compte, tu ne peux ni voir le catalogue ni télécharger.

Compte & abonnements

Les paiements et la gestion d’offre se font sur le site (espace client), pas dans l’app.

  • Gratuit — découverte limitée.
  • Standard — enregistrement mods, export PNG.
  • Pro — 4K + outils IA.
  • Premium — IA avancée + support prioritaire.

Voir les tarifs ou ton espace client.

FAQ

L’app demande un compte

Oui pour synchroniser l’offre et la bibliothèque. Sans auth configurée en dev, certaines portes restent ouvertes.

Mes fichiers restent-ils en local ?

Oui. Le décodage et le rendu sont locaux. Seuls compte, abonnement et bibliothèque cloud passent par Supabase.

Où gérer Stripe ?

Dans le dashboard web → Facturation (portail Stripe client).
